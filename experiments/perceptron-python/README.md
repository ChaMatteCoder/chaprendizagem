# Perceptron em Python

Implementação de apoio para o módulo de Perceptron. Estes arquivos preservam a lógica original do treinamento e podem servir como referência para futuras integrações com a interface React.

## Arquivos

- `dados.py` define as matrizes dos dígitos e monta a base de treinamento.
- `perceptron.py` implementa um perceptron binário.
- `rede_perceptron.py` combina dez perceptrons em uma estratégia um-contra-todos.
- `main.py` executa o treino e imprime os resultados.

## Execução

```bash
python main.py
```
